# DineSmart Restaurant Recommender(Prolog)
A knowledge based restaurant recommendation system using Prolog.

## Features
-Restaurant database with 30+ restaurants
-Budget, distance and rating filtering
-Cuisine based search
-Restaurant profile viewer
-Rating recommendation system
-Interactive menu system

## Technologies
Prolog
SWI-Prolog

## How to Run
1. Install SWI-Prolog
2. Load main.pl

1. ?- consult(main). 
2. ?- [main].
3. ?- menu_loop.

## Example Queries
View full restaurant profile  
Get filtered recommendations  
Search by cuisine
